Для запуска проекта выполните следующие действия:

1) Поместите папку localhost в рабоий каталог для доменов фреймворка laravel на вашем сервере (если там уже есть аналогичная папка, перед этим удалите ее)
2) Выполните миграции для базы данных с пробными данными или импортируйте базу из файла metrica_db_first_test.sql
3) В файле metrica_test.js в единственном параметре модуля добавьте имя домена сайта-админки на вашем сервере
4) Поместите файд metrica_test.js на сайт, который хотите отслеживать.
5) Зарегистрируйтесь на сайте-админке, войдите по паролю и логину( если импортировали базу из файла metrica_db_first_test.sql 
	тот регистрироваться не надо, просто введите 
		логин: test_user_1@mail.ru 
		пароль: test_user_1 ).
6) Для добавления нового сайта к отслеживанию, 
	авторизуйтесь на сайте-админке, 
	добавьте url сайта в список сайтов, 
	разместите скрипт metrica_test.js на нужном сайте (на каждой странице, с которой будете собирать данные)
